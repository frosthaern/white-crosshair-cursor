# white-crosshair-cursor

A minimal white crosshair cursor theme for Linux.

## Installation

1. Clone this repository to your local icons directory:
```bash
git clone <repository-url> ~/.icons/white-crosshair-cursor
```

2. Set the cursor theme in your configuration:
   - For **Sway/Wayland**, add to your `~/.config/sway/config`:
     ```
     seat seat0 xcursor_theme white-crosshair-cursor
     ```
   - For **other environments**, use `white-crosshair-cursor` as the theme name in your cursor settings

## Compatibility

This cursor theme has been tested and confirmed working on:
- Sway (Wayland)

Compatibility with other desktop environments and window managers is not guaranteed but may work. Use at your own discretion.

## How It Was Made

This cursor theme was created by:
1. Downloading cursor icons from [Icons8](https://icons8.com)
2. Converting the images to xcursor format using standard Linux cursor tools
